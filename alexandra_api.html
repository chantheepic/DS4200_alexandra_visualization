Getting Started with the Hotel Alexandra Web API

For ease of data access, a simple web api has been created to aid data retreival
if there are any functionality additions you would like, please send them by email.

Call Structure


<script>
    $.get("https://chanminis.online/public/api/alexandra/trafficflow/time/07:00/17:00", function (result) {
        console.log(result);
    });
</script>

https://chanminis.online/public/api/alexandra/trafficflow/time/07:00/17:00
The following returns a json of all traffic flow data between 7am and 5pm. A jQuery AJAX call is made via the $.get function and the URI is given as an argument. Use a callback function to ensure that the call is completed before the data is displayed. Clicking on the link above shows the JSON response.

Supported URIs

Note that all calls should start with https://chanminis.online/public/api/alexandra
Send me an email if something doesn't work

URI supported commands
Traffic Flow
/trafficflow/time/{startTime}/{endTime} Retreive all traffic flow seperated by type between {startTime} and {endTime} time selection is done by 15 minute increments. (14:30 works but 14:31 does not)
Data exists from 7am to 5pm both inclusive.
/trafficflow/timetype/{startTime}/{endTime}/{type} Retreive all traffic flow between {startTime} and {endTime} only of type {type} time selection is done by 15 minute increments. (14:30 works but 14:31 does not)
Data exists from 7am to 5pm both inclusive. Queryable types are CARS, HV, PAB
/trafficflow/timetotal/{startTime}/{endTime} Return sum of vehicle volume between {startTime} and {endTime} seperated by type time selection is done by 15 minute increments. (14:30 works but 14:31 does not)
Data exists from 7am to 5pm both inclusive.
